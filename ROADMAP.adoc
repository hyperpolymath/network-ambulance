==== Repair Modules

[cols="1,1,3"]
|===
|Module |Status |Capabilities

|*DNS Repairs*
|Complete
|Create/fix resolv.conf, add DNS servers, reset to defaults, restart systemd-resolved, flush cache, configure NM DNS

|*Interface Repairs*
|Complete
|Bring up interfaces, restart interfaces, renew DHCP, reset to DHCP, repair down interfaces

|*Routing Repairs*
|Complete
|Add default route, remove duplicates, repair gateway, flush & rebuild routing table

|*NetworkManager Repairs*
|Complete
|Restart service, reconnect connections, enable interface management, resolve conflicts, reset connections
